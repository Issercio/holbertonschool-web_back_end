# Dossier RNCP 5

Ce dossier sert a preparer ton dossier RNCP 5.
On va le completer pas a pas selon tes instructions.

## Contraintes officielles

- Rapport RNCP 5: entre 30 et 50 pages (hors page de garde, sommaire et annexes).
- Le dossier doit couvrir les 2 activites du referentiel:
	- Developper la partie front-end d une application web ou web mobile securisee.
	- Developper la partie back-end d une application web ou web mobile securisee.

## Structure

- `00_admin/` : documents administratifs
- `01_profil/` : presentation personnelle, parcours et Dossier Professionnel (DP)
- `02_projets/` : fiches projets (rapport/memoire)
- `03_competences/` : mapping competences RNCP
- `04_preuves/` : captures, attestations, livrables
- `05_annexes/` : annexes diverses

## Les 4 preparations non-techniques RNCP 5

1. Preparation initiale (cadrage)
2. Rapport / memoire de projet (30-50 pages)
3. **Dossier Professionnel (DP)** - RNCP 37674 DWWM → `01_profil/dossier_professionnel_DP.md`
4. **Support de presentation orale** → `02_projets/support_presentation_orale.md`

## Methode de travail

1. Tu m envoies les details petit a petit.
2. Je range et redige directement dans les bons fichiers.
3. Je te confirme a chaque ajout.

## Contexte projet en cours

- Projet principal declare: mise a jour d un site de vente (fleur, parfum et autres produits).
