# Personal Data

This project focuses on handling personal data securely, including:
- Obfuscating PII (Personally Identifiable Information) in log messages
- Using regex for data filtering
- Implementing secure logging practices

## Learning Objectives
- How to obfuscate PII fields in log messages
- How to use regex for pattern matching and substitution
- Understanding data privacy and security best practices

## Requirements
- All files interpreted/compiled on Ubuntu 20.04 LTS using python3 (version 3.9)
- All files should end with a new line
- First line of all files should be `#!/usr/bin/env python3`
- Code should use pycodestyle style (version 2.5)
- All files must be executable
- All modules, classes, and functions must have documentation
- All functions should be type annotated
