# 0x0B. Redis basic

This project covers basic usage of Redis with Python, including decorators for call counting and call history, and a replay function to display method call history.

## Features
- Store and retrieve data in Redis
- Count method calls using decorators
- Store and display call history (inputs/outputs)
- Fully documented and type-annotated code

## Usage
- All code is in `exercise.py`
- Example usage and tests are provided in the project directory

## Requirements
- Python 3.9+
- Redis server running
- `redis` Python package (`pip install redis`)
